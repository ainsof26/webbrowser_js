• The first property of open() is the url of a document loaded in a new window.
  • The second property of open() is the name of a new window. '_self' means the same window where the script is executed.
  • '_blank' means a new different window.
  • You can put a name on the window.
  • All the 'open()'s which have the same name property open new document on the same window .
  • The third property involves the shape of the window.
